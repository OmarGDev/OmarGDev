## 🚀 Sobre Mí

Desarrollador backend enfocado en el ecosistema de **Java** y **Spring Boot**. Me apasiona diseñar y construir **APIs REST** robustas, implementar arquitecturas limpias por capas y asegurar la integridad de los datos.

*   🧠 **Mi Enfoque:** Desarrollo de sistemas robustos, mantenibles y eficientes.
*   🔐 **Seguridad:** Especial interés en autenticación, autorización y protección mediante **JWT** y **Spring Security**.
*   🗄️ **Persistencia:** Modelado, migración y optimización en bases de datos relacionales.
*   🛠️ **Calidad:** Comprometido con las buenas prácticas, Clean Code y patrones de diseño.
